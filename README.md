# Mayhem Belt - GMTK 2025

This is our submission to the [2025 GMTK GameJam](https://itch.io/jam/gmtk-2025), a competition where you create an entire game in 4 days.  The game can be played [here](https://jtabor.itch.io/mayhem-belt).

Controls:

`W/S` - Forward Back

`A/D` - Rotate L/R

`Q/E` - Strafe L/R

`Shift` - boost

`C` - brake (slows movement on all axis)

`RMB` - free look

`Scroll` - zoom

`G` - Blink (teleport) - hold to see where you'll blink to, release to blink

Tractor Beam - Click an asteroid you want to attract, press `T` to activate the tractor beam and the tractor beam ball will attract the object if it's in range. 

`]` - Skip level - There are only two levels, but if your game crashes on the second one, you can skip the first one when you reload the page.
